```javascript
async function startServer() {
  try {
    // Ensure uploads directory exists
    const uploadsDir = path.resolve(__dirname, '../uploads');
    fs.mkdirSync(uploadsDir, { recursive: true });
    
    // ‚≠ê RUN DATABASE MIGRATIONS (AUTOMATIC!)
    await migrationRunner.runMigrations();
    //    ^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^
    //    This runs BEFORE server starts
    
    // Start the server
    app.listen(PORT, () => {
      console.log(`Server running on port ${PORT}`);
    });
  } catch (error) {
    console.error('Failed to start server:', error);
    process.exit(1);  // If migrations fail, server won't start
  }
}

startServer();  // Called immediately on backend start
```

### Key Points

1. **No manual intervention needed** ‚úì
2. **Runs before server starts** ‚úì
3. **Fails gracefully** - if migration fails, server won't start ‚úì
4. **One-time execution** - won't run same migration twice ‚úì
5. **Transaction-safe** - rollback on error ‚úì
