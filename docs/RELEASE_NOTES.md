# Release Notes

## 2026-03-06

### End-of-Season Roster Progression

- Added an **Advance Season** action on the Recruiting page to run end-of-season processing in one step.
- Committed recruits are now moved from the Recruiting board onto the roster as incoming freshmen.
- Existing roster players now advance to the next class year during season rollover.
- Added redshirt tracking to player records:
  - New **Redshirt this player for the current season** checkbox on add/edit player workflows.
  - Redshirted players do not advance during rollover.
  - Once a player uses a redshirt season, they cannot be redshirted again.
- Graduating players are moved off the active roster depth chart and are now viewable on a new **Graduates** screen.
